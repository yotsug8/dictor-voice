Инструкция по установке

Голос Диктора

Меняет ваш голос на голос диктора в реальном времени и отправляет его в голосовой чат — вы говорите, а собеседники в игре или звонке слышат чистый синтезированный голос.

Только Windows Русская речь Нужен VB-Cable (бесплатно)
01

Установка за три шага

Порядок важен: без VB-Cable вас никто не услышит.

1

Скачайте программу с GitHub

Есть видеокарта NVIDIA → возьмите NVIDIA-версию (быстрее): на странице проекта на GitHub нажмите зелёную кнопку «Скачать NVIDIA» — откроется Google Drive с файлом Diktor-NVIDIA_….zip.

Нет видеокарты NVIDIA или не уверены → возьмите CPU-версию (работает у всех, медленнее): на GitHub откройте раздел Releases и скачайте файл Diktor_CPU_….zip (в разделе Assets последнего релиза).

Распакуйте архив в любую папку и не вынимайте Diktor.exe из неё.

2

Установите VB-Cable и перезагрузите ПК

Это бесплатный «виртуальный кабель», по которому голос диктора попадёт в игру. Скачайте с vb-audio.com/Cable, запустите установщик от имени администратора и обязательно перезагрузите компьютер.

Важно: без перезагрузки Windows не увидит новый микрофон, и программа сообщит, что VB-Cable не найден.
3

Запустите Diktor.exe

При первом запуске программа докачает модель распознавания речи из интернета — нужно подключение к сети. Дальше интернет требуется только для перевода.

02

Как настроить звук

Всего два выбора — куда программа отдаёт голос и откуда его берёт игра.

🎤Ваш микрофон
→
🎙️«Голос Диктора»
меняет голос
→
🔌VB-Cable
(кабель)
→
🎮Игра / Discord
→
🗣️Вас слышат
1. В программе «Голос Диктора»

Вкладка «Устройства» → поле «Куда выводить звук», выберите:

CABLE Input
2. В игре, Discord или звонилке

Настройки звука → микрофон, выберите:

CABLE Output (VB-Audio)

Это два конца одного кабеля VB-Cable: программа отдаёт голос в «вход» (CABLE Input), а игра берёт его из «выхода» (CABLE Output). Свой обычный микрофон программа слушает сама — системный по умолчанию из параметров звука Windows, отдельно выбирать не нужно.

03

Как пользоваться

После настройки — три действия.

1

Выберите голос и модель

На вкладке «Голос» — диктор (мужской/женский), точность распознавания (base — быстрее, medium — точнее) и скорость речи. Без видеокарты выбирайте tiny или base, иначе будет заметная задержка.

2

Нажмите «Старт» и проверьте кнопкой «Тест»

«Тест» проиграет проверочную фразу в выбранное устройство. Если звук идёт в CABLE Input, в самой программе вы его не услышите — это нормально, его слышат собеседники. Чтобы проверить на слух, временно выберите свои наушники.

3

Говорите

После короткой паузы диктор озвучит сказанное. F8 — заглушить диктора (в чат идёт тишина, а не ваш голос), F9 — старт/стоп. Обе клавиши работают, даже когда окно свёрнуто.

04

Если тебя не слышно

Самое частое — и что с этим делать по порядку.

Собеседники ничего не слышат

В игре, Discord или звонилке микрофоном должен стоять CABLE Output (VB-Audio), а в самой программе вывод — CABLE Input. Проверьте оба конца.

VB-Cable не найден

Установите его от имени администратора и перезагрузите ПК, затем нажмите ↻ рядом со списком устройств в программе.

Программа не распознаёт мою речь

В параметрах звука Windows выберите нужный микрофон устройством ввода по умолчанию — программа слушает именно его. Во время работы (после «Старт») индикатор уровня на вкладке «Устройства» должен реагировать на голос.

Голос диктора молчит или запаздывает

Для синтеза и перевода нужен интернет. Без видеокарты тяжёлая модель (medium) работает медленно — выберите tiny/base. Громкость не должна стоять на 0%.

Голос персонажа (RVC) звучит обычным голосом

Кастомные голоса персонажей работают только с видеокартой NVIDIA. Без неё программа озвучивает обычным голосом диктора — это ожидаемо, не поломка.

Антивирус ругается на Diktor.exe

Ложное срабатывание — так помечаются почти все программы, собранные через PyInstaller. Добавьте файл в исключения или соберите программу сами из открытого исходного кода.

05

Что ещё умеет

Профили голоса

Сохраните сочетание голоса, скорости, перевода, громкости и тона под именем и переключайтесь одним выбором.

Озвучка текста

Введите фразу в поле внизу и нажмите Enter — диктор произнесёт её, даже если вы молчите.

Переводчик-диктор

Выберите язык в списке «Перевод» — диктор произнесёт вашу речь на нём (нужен интернет).

Голоса персонажей (RVC)

Положите модель .pth в папку voices/ — голос появится с пометкой «[Персонаж]» после перезапуска программы. Только NVIDIA.

Поверх всех окон

Окно не теряется за игрой, а крестик сворачивает программу в значок у часов.

Тон голоса персонажа

Ползунок меняет тембр на лету, без перезагрузки модели. Действует только на голоса персонажей (RVC), то есть в NVIDIA-версии.
