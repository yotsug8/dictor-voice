# Голос Диктора

Замена голоса на синтезированный голос диктора в реальном времени (русский). Для Windows.
Говоришь — после паузы программа распознаёт речь и озвучивает её голосом диктора,
выводя звук в виртуальный микрофон, чтобы тебя слышали в голосовом чате.

## Скачать

- **Видеокарта NVIDIA** → [Diktor-NVIDIA.zip](https://drive.google.com/file/d/1_v0HTdAV5YYG06lYsdi6A3oDLG3irwDX/view?usp=drive_link) (быстрее, с Google Drive)
- **Без видеокарты NVIDIA** → `Diktor-CPU.zip` в разделе [Releases](https://github.com/yotsug8/dictor-voice/releases) (работает у всех, медленнее)

> При скачивании NVIDIA-версии Google Drive может предупредить «не удалось проверить на вирусы» (из-за размера) — это нормально, нажмите «Скачать всё равно».

## Установка

1. Скачай нужный zip и распакуй в любую папку.
2. Установи **VB-Cable**: <https://vb-audio.com/Cable/> — запусти установщик от имени администратора и **перезагрузи ПК**.
3. Запусти `Diktor.exe` из распакованной папки (не вынимай его из неё).

## Как пользоваться

1. Выбери голос диктора, точность модели (`base` — быстрее, `medium` — точнее) и скорость речи.
2. В поле «Куда выводить звук» выбери **Виртуальный микрофон (CABLE Input)**.
3. Нажми **Старт**. При первом запуске докачается модель распознавания (нужен интернет).
4. В приложении/игре выбери микрофон **CABLE Output (VB-Audio)**.
5. Говори — после паузы диктор повторит твоими словами.

## Возможности

- **Озвучка текста** — введи текст в поле внизу и нажми Enter (или «Озвучить»): диктор произнесёт его, даже если не говоришь в микрофон.
- **Переводчик-диктор** — в списке «Перевод» выбери язык (английский, немецкий и др.), и диктор будет произносить твою речь на этом языке.
- **Пауза звука** — кнопка «🔊» или клавиша **F8**: временно говорить обычным голосом.
- **Старт/стоп** — клавиша **F9**, работает даже когда окно не активно.
- **Индикатор микрофона** — полоса в окне показывает, что физический микрофон слышит звук.
- **Громкость и скорость** диктора настраиваются в окне.
- **Поверх всех окон** — окно не теряется за приложением.
- **Сворачивание в трей** — крестик прячет окно в значок у часов; выход — через значок (правый клик → «Выход»).
- **Кастомные голоса персонажей (RVC)** — необязательно: положи модель голоса в папку `voices/`
  (см. `voices/README.txt`). Требует сборки из исходников и видеокарты NVIDIA — подробности в [BUILD.md](BUILD.md).
  В готовый .exe из раздела «Скачать» эта функция не входит.

## Ограничения

- Есть задержка: речь → пауза → озвучка (несколько секунд).
- Перевод работает через интернет и добавляет небольшую задержку.
- CPU-версия заметно медленнее, чем NVIDIA.
- Антивирус может выдать ложное срабатывание.

## Если не работает

- Не установлен VB-Cable → поставь его и перезагрузи ПК.
- Скачал NVIDIA-версию, но видеокарты NVIDIA нет → скачай CPU-версию.
- Нет интернета при первом запуске → модель не скачается, подключись к сети.
- Звука нет у собеседников → в приложении/игре выбери микрофон **CABLE Output (VB-Audio)**.

## Безопасность

Программа не содержит вредоносного кода. Все библиотеки берутся из официальных
источников (PyPI, PyTorch), а полный исходный код открыт в этом репозитории.

- **Антивирус может ругаться.** Это ложное срабатывание: так помечаются почти все
  программы, собранные через PyInstaller. Реальной угрозы нет, файл можно добавить
  в исключения.
- **Хотите убедиться сами** — проверьте файл на [VirusTotal](https://www.virustotal.com)
  (несколько эвристических срабатываний для таких сборок — это норма).
- **Не доверяете .exe** — соберите программу сами из исходников по инструкции
  [BUILD.md](BUILD.md). Так вы получите тот же результат, но из проверенного вами кода.
