Папка для кастомных голосов (RVC) — голоса персонажей.

Как пользоваться:
1. Скачайте модель голоса с weights.gg или AI Hub — это файл .pth
   (часто рядом идёт файл .index — кладите оба сюда, индекс улучшает качество).
2. Положите файлы в эту папку voices/.
3. Перезапустите программу — голос появится в списке «Голос диктора»
   с пометкой [Персонаж] и именем файла.

Что происходит при выборе голоса [Персонаж]:
речь -> текст -> базовый голос (Edge TTS) -> RVC накладывает тембр персонажа -> вывод в виртуальный микрофон.

ТРЕБОВАНИЯ:
- Установленный пакет rvc-python (см. BUILD.md).
- Видеокарта NVIDIA — на процессоре конверсия слишком медленная для
  реального времени.
- При самом первом запуске RVC докачивает базовые модели (HuBERT, RMVPE)
  из интернета — нужен интернет и ~400 МБ места.

Если rvc-python не установлен или нет видеокарты — программа просто озвучит базовым голосом и напишет об этом в лог, ничего не сломается.
